## 사용법

1. 원하는 AI의 `setup-guide.md`를 열고 단계별로 따라하기
2. AI가 동작하면 `auto-setup.md` 내용을 첫 대화에 붙여넣기
3. AI가 질문하면 답하기 → 세팅 끝
